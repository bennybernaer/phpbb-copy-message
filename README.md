# Copy Message Text

A small phpBB 4.0 extension for prosilver that adds a one-click copy button above message textareas.

## Features

- Works in Quick Reply, the full posting editor and private messages.
- Copies the complete message text to the clipboard.
- Shows a check icon briefly after copying.
- Keeps the full textarea writing width available.
- Supports dynamically opened Quick Reply editors.

## Requirements

- phpBB 4.0.x
- PHP 8.2 or newer
- prosilver style

## Installation

1. Upload the extension to `ext/bennybernaer/copymessage/`.
2. Open the ACP.
3. Go to **Customise > Manage extensions**.
4. Enable **Copy Message Text**.
5. Purge the phpBB cache.
